Classes - getter and setter

1. get and set is not a function. its a property, its used to assign or get a value.
2. get() must no param
3. set(value) must param

class Person{ constructor(name, age){ this.name = name; this.age = age; } greet(){ console.log(`Hello ${this.name}!`); } static info(){ console.log("information"); } //get name() is not allowed, because its already declared in property; get Name(){ return this.name; } set Name(value){ this.name = value; } } const person_1 = new Person("Lmj", 28); person_1.greet(); console.log(`Before setter ${person_1.Name}`); person_1.Name = "Suhail"; console.log(`After setter ${person_1.Name}`);

Classes - Extend

if class extends another class the constructor inside must have the super constructor. super constructor call the parent constructor.

class Person{ constructor(name, age){ this.name = name; this.age = age; } greet(){ console.log(`Hello ${this.name}!`); } static info(){ console.log("information"); } //get name() is not allowed, because its already declared in property; get Name(){ return this.name; } set Name(value){ this.name = value; } } class Developer extends Person { constructor(name, age, projects){ super(name, age); this.projects = projects; } get Projects(){ return this.projects; } set Projects(value){ this.projects = value } } const person_1 = new Person("Lmj", 28); person_1.greet(); const person_2 = new Developer("Suhail", "2", [{id : 11, project :"ES6"}]); console.log(person_2.Projects)
