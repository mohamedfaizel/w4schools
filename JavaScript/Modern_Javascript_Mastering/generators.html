Generators

function* keyword used to create a generators

function* listContacts(){ yield "Mohammed"; yield "Faizel"; yield "Suhail"; yield "Muttaqin"; yield "Lmj"; } const contacts = listContacts(); console.log(contacts.next()) {value: "Mohammed", done: false} // done false mean Loop is not end

Generators - Example

Perfect usecase for Generator a syncronous task in series. if you want to run synchronous task in waterfall methods 3rd task depend on 2nd task and 2nd task depend on 1st task.

function networkRequest(url){ fetch(url).then(data => data.json()).then(data => dataGen.next()) } function* asyncTask(){ console.log("fetching beers"); const beers = yield networkRequest("http://api.react.beer"); console.log(beers); console.log("fetching haider"); const wes = yield networkRequest("http://api.github.haider"); console.log(wes); console.log("fetching Joe"); const joe = yield networkRequest("http://api.discogs.com"); console.log(joe); } const dataGen = asyncTask(); dataGen.next() //start the generator

Looping in Generators

for of loop also used insted .next() in every time

function* listContacts(){ yield "Mohammed"; yield "Faizel"; yield "Suhail"; yield "Muttaqin"; yield "Lmj"; } const contacts = listContacts(); for(let name of contacts){ console.log(name); }
