Set

1. Sets is like array, Sets do not allow the duplicates. 2. Iterates (.next()) is possible 3. methods : add(), delete(), has(), size(),clear(), values(), next()

const persons = new Set(); persons.add("Lmj"); persons.add("Suhail"); persons.add("Muttaqin"); persons.has("Lmj"); persons.delete("Lmj"); const iter = persons.values(); console.log(iter.next()); console.log(iter.next()); for(p of persons){ console.log(p); } // Direct values const developers = new Set(["Lmj", "Mohamed", "Faizel", "Suhail", "Muttaqin"]); console.log(developers.size()); //dynamic Values const students = ["S1", "S2", "S3", "S4"]; const studentSet = new Set(students); console.log(studentSet.size()); studentSet.clear(); console.log(studentSet.size()); {value: "Suhail", done: false} {value: "Muttaqin", done: false} Suhail Muttaqin
