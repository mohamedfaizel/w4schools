For of - entries

* entries make a every array value to object (key is index);
* we can iterator the value using next method
Example
const iterator = friends.entries();
console.log(iterator.next()); // first object value print
console.log(iterator.next()); // second object value print

let friends = ["Lmj", "Faizel", "Suhail", "Muttaqin"]; for(let [index, friend] of friends.entries()){ console.log(`key :${index} value : ${friend}`); } key :0 value : Lmj key :1 value : Faizel key :2 value : Suhail key :3 value : Muttaqin
