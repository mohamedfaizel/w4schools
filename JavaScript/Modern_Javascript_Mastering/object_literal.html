Object Literal - create Techniq

in Es6 allows both the key and value variable are same, then remove the value variable. we can also rename the variable.

const first = "Lmj"; const last = "Mohammed"; const age = 29; const phone = 9600467899 let person = { firstname : first, last, age, phone } console.log(person); {firstname: "Lmj", last: "Mohammed", age: 29, phone: 9600467899}
