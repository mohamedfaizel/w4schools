Visual Studio Code Exxtensions

Install this Extenstion to your Visual studio Code
  • Angular Essentials (John Papa)
  • Prettier - Code formatter
    • Prettier used to format your code automatically when we save file.
    • Create a file ".prettierrc"
  • "cascadia-code" - Font
    • Download and install cascadia font in your system.
    • https://github.com/microsoft/cascadia-code/releases
    • VS Code => Go to setting ctrl + ,
    • Search Font Family
    • 'Cascadia Code' = add this in front of the line. Note : Quote is important.
    • Search Font Ligatures (For Styling Codes Eg : ==, !=)
    • Enable font ligatures
  • File Exclude - Hide folder / files in side menu window.
    • VS Code => Go to setting ctrl + ,
    • Search "File Exclude"
    • Click "Add Pattern"
    • Enter File name or Folder name
    • Click "ok"
  • Zen Mode : Center Layout
    • untick this option for Zen Mode code window not show in center.

Visual Studio Code - Shortcut


  • Ctrl + , Settings
  • Ctrl + shift + p command prompt
  • F5 Program run in Debug mode
  • Ctrl + Z Zen Mode
  • Ctrl + ` Hide / Show Side Menu window
  • Ctrl + B Hide / Show Terminal window
  • Ctrl + Shift + F Global Search
