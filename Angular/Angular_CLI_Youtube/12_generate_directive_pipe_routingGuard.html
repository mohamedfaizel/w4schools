generate directive

  • ng generate directive directiveName or ng g d directiveName => To generate a directive.
    Note :more than one module it will makes error More then one module matches
    • ng g d directiveName --skip-import => Do not attempt to import this directive into any module. Later we can mannually imported to where we needed
    • ng g d directiveName -m=app.module => To generate a directive and import it into module
  • ng g m moduleName -m=app.module => To generate a module and import it into anothher module

generate pipes and routing guard

ng generate pipe pipeName or ng g p pipeName
ng generate guard guardName or ng g g guardName Note : it does not make an error if we have more then one module also, because it doesn't try to import or register anything.
