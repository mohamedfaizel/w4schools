angular-cli.json

The setting from the angular cli configuration files are used when we
  • Generates angular features like, component, pipes, services etc
  • Run unit and ene2ent test
  • Build the application etc
Note : The value in the angular cli configuration file depends on the options that you have used with the ng new command when generating a new angular projects
Some of these options like --prefix can be used with several command ng new and ng generate
