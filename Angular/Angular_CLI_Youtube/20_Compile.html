Compile Angular App

ng serve
  • compile and serve the application from memory
  • Does not write the build file to the disk
  • Typically used to run the applicatioon local development machine
  • Cannot be used for deploying the build to another server(Ex : Testing, staging or Production server)
ng build
  • Compiles the application to the "dist" folder.
  • Can be used to produce both development and production builds.
  • Typically used to deploy the application on another server.
  • ng build or ng build --dev => development build
  • ng build --prod => production builds
Production Vs Development build
  • Production build file sizes are significantly less then develoment build file sizes
  • production file does not generate source map files
  • Production build extract css from global styles onto css files instead of js
