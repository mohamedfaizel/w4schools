Compiler

In Angular we have 2 models of compiler
  • JIT - Just in time compiler
  • AOT - Ahead of time compiler
JIT VS AOT
  • JIT => Compiles the application just in time in the browser at run time
  • AOT => Compiles the application at build time
  • JIT => For jit compilation the browser needs to download the angular compiler
  • AOT =>AOT Compilation does not have to
  • JIT => the applicatiojn is beign JIT compiled in the browser, User have to wait
  • AOT => the application is pre compiled so there is no such wait
  • JIT => template binding errors are only know at run time
  • AOT => will come to know about them at build time
By defaultthe following 2 commands use jit compilation
  • ng build
  • ng serve
with either of thecommand we can use --aot option to turn on AOT
  • ng build --aot true
  • ng serve --aot true
To turn off AOT for the production build, set --aot option to false
  • ng build --prod --aot false
