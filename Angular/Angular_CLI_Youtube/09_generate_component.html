generate component

  • you must have the npm packages installed to be able to generate components using Angular cli.
  • Use npm install command to install the packages
  • ng generate component componentName or ng g c componentName => To generate a component.
  • ng g c abc/xyz => To place the generated component folder in a different folder other then "app"
  • ng g c abc --flat or ng g c abc/xyz --flat => To create a component withou a folder, use --flat option.
  • ng g c abc --dry-run or ng g c abc -d => use --dry-run flags to see the list of files and folders that will be generated
  • ng g c abc -it -is --spec=false => To create a component with an inline HTML and Styles. Note : --spec=false spec file also not generated.
  • ng g c --style=scss => To use SCSS instead of CSS with your component
