ng serve

  • ng serve => compile and serve the application
  • ng serve --open => compile the application and open the application in browser
  • ng serve --help => get the help text
  • ng serve --watch true => Run build when file changes.
  • ng serve --live-reload true => wheather to reload the page on changes.
  • ng serve --port 8288
  • ng serve --port 8288 --extract-css trueExtract css into css file instead of JS.
