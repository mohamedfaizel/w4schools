Customize command prompt

  • To increae the screen buffer size, increase the height property value under the Screen buffer size section on the layout tab in the property window
  • ng --help => view all the angular helps
  • ng --help | clip => To redirect the output of a command to windows clipboard, we can paste it into anywere like word, notepad...
  • c:\>ng --help > Help.txtTo redirect the output of a command to a text document or word Document(.doc)
  • from the properties window of the command prompt we can also change the cursor size, font size, color and background color
