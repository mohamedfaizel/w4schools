generate module

  • ng generate module moduleName or ng g m moduleName => To generate a module.
    Note :mannually we import file and imports this module in app.module.ts
  • ng g m moduleName -m=app.module => To generate a module and import it into anothher module
  • ng g m moduleName --flat=true => if you do not want to dedicated folder for the module use --flat
  • ng g m moduleName --spec=true => To create a module with spec file.
