Routing Mannually with separate file

Part-1 :
  1. ng new employeeManagement --routing => Create new project with separate routing files with basics codes
  2. ng g c home --module=app-routing.module => Create new Component - home
  3. ng g c employees --module=app-routing.module => Create new Component - employees
  4. ng g c pageNotFound --module=app-routing.module => Create new Component - pageNotFound
Using the above commands the Angular CLI automatically do all the small stuff
const routes: Routes = [ { path: '', redirectTo: '/home', pathMatch: 'full' }, { path: 'home', component: HomeComponent }, { path: 'employees', component: EmployeesComponent }, { path : '**', component: PageNotFoundComponent } ]; export class AppRoutingModule {} export const AppRoutingComponents = [HomeComponent, EmployeesComponent, PageNotFoundComponent]; //this is good practise for add the component in array i) import { AppRoutingModule, AppRoutingComponents } from './app-routing.module'; ii) declarations: [ AppComponent, AppRoutingComponents ], imports:[AppRoutingModule]
