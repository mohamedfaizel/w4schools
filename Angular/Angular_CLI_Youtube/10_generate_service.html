generate services

  • generate Services is very similar to generating components
  • ng generate service serviceName or ng g s serviceName => To generate a service.
  • ng g s employee -m=app.module => To register the newly generated service with the specified module.
  • ng g s abc --flat=false => To create a services with a seprate new folder
  • ng g c abc --dry-run or ng g c abc -d => use --dry-run flags to see the list of files and folders that will be generated
  • ng g s abc --spec=false => To create a services without spec file.
