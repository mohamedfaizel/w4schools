Component

Component is a collection of Template + class + metadata
  • Template represent view : This is created using HTML
  • Class is nothing but Code : This is created using TypeScript. it also contains Data & methods to control View Logic
  • Metadata : This is Information that angular needs to decide, Particular Class Infact angular Component or just a regular class.
    Decorator : Just a function provides the information about the class attached to it. Components have component Decorator.

Component

app.component.ts

metadata (@Component) : its Contains both metadata and templates

app.components is the main Componetnts all others comes under the main component only.>
  • Selector : This indicates where the template(HTML + CSS) will placed in index.html
    • custom-tag : 'app-root' => <app-root> .... </app-root>
    • class name : '.app-root' => <div class="app-root"> .... </div>
    • attribute : '[app-root]' => <div app-root> ..... </div>
  • Template / TemplateUrl
    • TemplateUrl : './test.component.html'
    • Template : '<div> Inline Templat </div>'
    • Template : `<div> MultiLine Template </div>`
  • StyleUrls / Styles : Its array
    • StyleUrls : ['./test.component.css']
    • Styles : [`div { color : red; }`]
import { Component } from '@angular/core'; //MetaData, its specific Component Decorator, the component Decorator attach function write below it(@Component). In our case AppComponet Class is the function for this. Its tell the Angular its not a regular class, Its Component. @Component({ selector: 'app-root', templateUrl: './app.component.html', styleUrls: ['./app.component.css'] }) export class AppComponent { //Its a class, AppComponent is the name of the class, its contain methods and property. title = 'hello-world'; }

Add new Component

command : ng g c test

g - generates, c - components, test - components name

Its generates the new componets and update the modules. we can explain it step by step.

  • Step 1 : its create a new forder test under src > app > test
  • Step 2 : test folder contains 4 files.
    1. test.component.css, 2. test.component.html, 3. test.component.spec.ts, 4. test.component.ts
  • step 3: app.module.ts will updated
    • import {TestComponent} from './test/test.component'
    • declarations: [AppComponent, TestComponent] // TestComponent is added
  • step 1, 2, 3 is automaticali done by Angular CLI
Implemenet test.Component in app.component.html
  • Step 1: get the selector name in test.component.ts (app-test)
  • Step 2: Open app.Component.html and create the custom tag app-test where you want to place the test.component.html
import { Component, OnInit } from '@angular/core'; @Component({ selector: 'app-test', templateUrl: './test.component.html', styleUrls: ['./test.component.css'] }) export class TestComponent implements OnInit { constructor() { } ngOnInit() { } }
