Angular - Tips

Tips to build a good Apps
  • Use Error Interceptors
  • Use route guards to prevent unauthorized users to access specific screens
  • Use Interfaces to create objects for data coming from the API
  • Use Components as much as possible
  • Route Resolvers instead of Safe Navigation operators
  • So data is got from the API by the service which then gives it to the resolver which then gives it to the component
  • Use hostlistener so that forms that take data inform the user if they are trying to navigate away from them without saving the information
  • Use BehaviourSubject observable to communicate between components that don't have a parent child relationship
  • Use the above to reduce network calls to the database
  • Use Reactive forms and Form builders with validators and custom validators for designing forms
  • Use time ago pipes for showing dates/last sent data/last activity etc
