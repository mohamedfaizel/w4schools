Shadow DOM Technology
How the angular css works. In generally if you define any css rule in anywhere or any stylesheet it will applicable for whole document. But how angular treat this ? Each component works differently. How ?
angular add a unique attribute to Each element in a component
  • Emulated => Default
    Unique attribute added in each Element in a component
  • None
    No attributes added, so its act alike a normal, which means if you add any css rule in any component css, it'll apply all the components
  • Native
    same as Emulated
@Component({ selector: 'app-server-element', templateUrl: './server-element.component.html', styleUrls: ['./server-element.component.css'], encapsulation: ViewEncapsulation.None // })
